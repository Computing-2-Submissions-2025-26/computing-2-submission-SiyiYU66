Namespace

Battleship

Description:

  • A pure-functional engine for the classic Battleship board game. The board is the single source of truth: a ship's identity and footprint are read directly from the cells (via cell.shipName), so every operation depends only on its arguments — there is no hidden module state to keep in sync.

(static) move_ship(board, ship_name, direction) → {Battleship.Board}

Description:

  • Shifts an already-placed ship one cell in the given direction. The vacated cells keep their shot flag (a hit there becomes a miss), while the cells the ship moves onto are reset to un-shot. Illegal moves (off-board, or blocked by a different ship) return the original board unchanged.
